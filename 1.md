## 🏗️ Kiến Trúc Tổng Quan

```
Client (Postman / VueJS)
        │
        ▼
  [Controllers]  ── nhận HTTP request, validate ModelState
        │
        ▼
  [Services]     ── business logic, conflict detection
        │
        ▼
  [DbContext]    ── EF Core ORM queries
        │
        ▼
  [PostgreSQL]   ── Railway hosted database
```

**Pattern áp dụng:**
- **Repository-like Service Layer** — Controllers không truy cập DB trực tiếp
- **DTO Pattern** — Tách biệt hoàn toàn Model (DB) với dữ liệu trả về API
- **Outbox Pattern** — Lưu domain events vào DB trước khi publish (EventPublisher)
- **Dependency Injection** — Tất cả service đăng ký Scoped qua interface

## 🗄️ Models (Entities)

### `Course.cs` — Khóa học

```csharp
public class Course
{
    public int Id { get; set; }
    [Required][MaxLength(200)]
    public string Name { get; set; }         // Tên khóa học
    [Range(0, double.MaxValue)]
    public decimal Fee { get; set; }          // Học phí (precision 18,2)
    [Range(1, 1000)]
    public int NumberOfSessions { get; set; } // Số buổi học
    [Required][MaxLength(50)]
    public string Level { get; set; }         // Beginner / Intermediate / Advanced
}
```

### `Class.cs` — Lớp học

```csharp
public class Class
{
    public int Id { get; set; }
    public int CourseId { get; set; }          // FK → Course (Restrict delete)
    public Course? Course { get; set; }
    [Required][MaxLength(200)]
    public string ClassName { get; set; }
    [Required][MaxLength(200)]
    public string TeacherName { get; set; }
    [Range(1, 1000)]
    public int MaxCapacity { get; set; }       // Sĩ số tối đa
    public int CurrentEnrollment { get; set; } // Sĩ số hiện tại (default: 0)
    [Required][MaxLength(50)]
    public string Status { get; set; }         // Opened / Closed / Cancelled
    public DateTime CreatedAt { get; set; }    // UTC timestamp
}
```

### `Classroom.cs` — Phòng học

```csharp
public class Classroom
{
    public int Id { get; set; }
    [Required][MaxLength(100)]
    public string RoomName { get; set; }  // Tên phòng, ví dụ: "Lab 101"
    [Range(1, 1000)]
    public int Capacity { get; set; }     // Sức chứa tối đa
}
```

### `ClassSchedule.cs` — Lịch học

```csharp
public class ClassSchedule
{
    public int Id { get; set; }
    public int ClassId { get; set; }       // FK → Class (Cascade delete)
    public Class? Class { get; set; }
    public int ClassroomId { get; set; }   // FK → Classroom (Restrict delete)
    public Classroom? Classroom { get; set; }
    public DayOfWeek DayOfWeek { get; set; } // 0=CN, 1=T2, ..., 6=T7
    [Required][MaxLength(50)]
    public string Shift { get; set; }        // Ca sáng / Ca chiều / Ca tối
    public TimeSpan StartTime { get; set; }
    public TimeSpan EndTime { get; set; }
}
```

### `OutboxEvent.cs` — Domain Event (Outbox Pattern)

```csharp
public class OutboxEvent
{
    public int Id { get; set; }
    [Required][MaxLength(100)]
    public string EventType { get; set; }  // e.g. "class.opened"
    [Required]
    public string Payload { get; set; }    // JSON serialized payload
    public DateTime CreatedAt { get; set; }
}
```

## 🗃️ Database Context

### `CourseDbContext.cs`

```csharp
public DbSet<Course> Courses { get; set; }
public DbSet<Classroom> Classrooms { get; set; }
public DbSet<Class> Classes { get; set; }
public DbSet<ClassSchedule> ClassSchedules { get; set; }
public DbSet<OutboxEvent> OutboxEvents { get; set; }
```

**Cấu hình quan hệ trong `OnModelCreating`:**

| Quan hệ | Hành vi xóa |
|---|---|
| `Class` → `Course` | `Restrict` — không xóa Course nếu còn Class |
| `ClassSchedule` → `Class` | `Cascade` — xóa Class thì xóa luôn Schedule |
| `ClassSchedule` → `Classroom` | `Restrict` — không xóa Classroom nếu còn Schedule |

**Precision:** `Course.Fee` → `decimal(18, 2)`

## 🌱 DbInitializer — Auto Migrate & Seed

Chạy tự động khi khởi động app trong `Program.cs`.

**Logic:**
1. Nếu có migration chưa apply → chạy `MigrateAsync()`
2. Nếu không có → `EnsureCreatedAsync()`

## ⚙️ Services

### `ICourseService` / `CourseService`

| Method | Mô tả |
|---|---|
| `GetAllAsync()` | Lấy toàn bộ khóa học |
| `GetByIdAsync(id)` | Lấy khóa học theo ID |
| `CreateAsync(dto)` | Tạo khóa học mới |
| `UpdateAsync(id, dto)` | Cập nhật thông tin khóa học |
| `DeleteAsync(id)` | Xóa khóa học — **ném `InvalidOperationException` nếu còn lớp học đang dùng** |

### `IClassService` / `ClassService`

| Method | Mô tả |
|---|---|
| `GetAllAsync()` | Lấy toàn bộ lớp học (Include Course) |
| `GetByIdAsync(id)` | Lấy lớp theo ID |
| `OpenClassAsync(dto)` | Tạo lớp mới → kiểm tra Course tồn tại → publish event `class.opened` |
| `GetClassesByTeacherAsync(name)` | Lọc lớp theo giáo viên (case-insensitive) |
| `EnrollStudentAsync(classId)` | Tăng `CurrentEnrollment` — **ném lỗi nếu đầy** |
| `UnenrollStudentAsync(classId)` | Giảm `CurrentEnrollment` — **ném lỗi nếu bằng 0** |

### `IScheduleService` / `ScheduleService`

| Method | Mô tả |
|---|---|
| `GetAllSchedulesAsync()` | Lấy toàn bộ lịch học |
| `GetSchedulesByClassIdAsync(classId)` | Lịch theo lớp |
| `GetSchedulesByTeacherAsync(name)` | Lịch theo giáo viên |
| `ConfigureScheduleAsync(dto)` | **Cấu hình lịch học với đầy đủ validation** |
| `RemoveScheduleAsync(id)` | Xóa lịch học |

#### Logic `ConfigureScheduleAsync` — Chi tiết validation:

```
1. Parse StartTime / EndTime (HH:mm → TimeSpan)
2. EndTime phải > StartTime
3. Kiểm tra Class tồn tại (KeyNotFoundException nếu không có)
4. Kiểm tra Classroom tồn tại (KeyNotFoundException nếu không có)
5. Class.MaxCapacity ≤ Classroom.Capacity (InvalidOperationException nếu vượt)
6. Kiểm tra trùng lịch (Overlap Detection):
   - Cùng DayOfWeek
   - Overlap time: NewStart < ExistingEnd AND NewEnd > ExistingStart
   - Cùng phòng OR cùng giáo viên
   → ném InvalidOperationException với message tiếng Việt chi tiết
7. Lưu ClassSchedule mới vào DB
```

### `IEventPublisher` / `EventPublisher`

**Outbox Pattern** — Khi `class.opened` event được phát:
1. Serialize payload thành JSON
2. Tạo bản ghi `OutboxEvent` lưu vào DB cùng transaction
3. Log thông tin event ra console để audit

```
PUBLISHED SYSTEM EVENT [class.opened] (Saved in Outbox Event Id: 5) @ 2026-06-14T07:30:00Z.
Payload: {"ClassId":1,"ClassName":"K01",...}
```

| Method | Endpoint | Mô tả | Response |
|---|---|---|---|
| `GET` | `/api/schedules` | Lấy tất cả lịch học | `200 OK` + `ScheduleDto[]` |
| `GET` | `/api/schedules/class/{classId}` | Lịch theo lớp | `200 OK` + `ScheduleDto[]` |

**HTTP Status codes đặc biệt cho `POST /api/schedules`:**
| HTTP Code | Trường hợp |
|---|---|
| `201 Created` | Tạo lịch thành công |
| `400 Bad Request` | Sai định dạng giờ / EndTime ≤ StartTime / Class vượt sức chứa phòng |
| `404 Not Found` | ClassId hoặc ClassroomId không tồn tại |
| `409 Conflict` | **Phát hiện lịch học bị trùng** (cùng phòng hoặc cùng giáo viên) |

## 🔗 Quan Hệ Database (ERD)

```
Course (1) ──────< Class (N)
                      │
                      │(1)
                      ▼
               ClassSchedule (N)
                      │
                      │(N)>──── Classroom (1)

OutboxEvent (độc lập — lưu domain events)
```

| Bảng | Khóa ngoại | Ràng buộc |
|---|---|---|
| `Class.CourseId` | → `Course.Id` | Restrict (không xóa Course nếu có Class) |
| `ClassSchedule.ClassId` | → `Class.Id` | Cascade (xóa Class → xóa Schedule) |
| `ClassSchedule.ClassroomId` | → `Classroom.Id` | Restrict |

---

## 🔒 Business Rules (Quy tắc nghiệp vụ)

| # | Rule |
|---|---|
| 1 | Không thể xóa `Course` nếu đã có `Class` đang sử dụng |
| 2 | Mở lớp (`OpenClass`) phải có `CourseId` hợp lệ |
| 3 | `CurrentEnrollment` không được vượt `MaxCapacity` |
| 4 | `CurrentEnrollment` không thể giảm dưới 0 |
| 5 | `Class.MaxCapacity` không được vượt `Classroom.Capacity` khi xếp lịch |
| 6 | Không xếp 2 lịch trùng giờ **cùng phòng** trong cùng ngày |
| 7 | Không xếp 2 lịch trùng giờ **cùng giáo viên** trong cùng ngày |
| 8 | Mỗi khi mở lớp, hệ thống publish event `class.opened` vào Outbox |
